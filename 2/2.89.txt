int x = random();
int y = random();
int z = random();

double dx = (double) x;
double dy = (double) y;
double dz = (double) z;

A. (float) x == (float) dx
1 т.к. int помещается в double

B. dx - dy == (double) (x - y)
0 т.к. x-y может переполниться

C. (dx + dy) + dz == dx + (dy + dz)
1 т.к. int помещается в double и сумма int'ов ассоциативна

D. (dx * dy) * dz == dx * (dy * dz)
0 т.к. произведение double не ассоциативно

E. dx / dx == dz / dz
0 <=> когда dz = dx = 0
