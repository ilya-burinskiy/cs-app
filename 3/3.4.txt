Assume than values of sp and dp stored in %rdi and %rsi.
First instruction shoud read from mem? do the appropriate conversion, and set
the appropriate portion of register %rax. The second instruction should then write
the appropriate portiion of %rax to memory. Recall that when performing a cast that
involves both a size change and a change of "signedness" in C, the operation should
change the size first.
